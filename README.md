The STM32-based Pedometer (Step Counter) System is a real-time embedded application developed using FreeRTOS and the MPU6050 motion sensor. The system continuously acquires accelerometer and gyroscope data, processes it using a step detection algorithm, and calculates the number of steps in real time. It is designed using a multitasking architecture, where separate tasks handle sensor data acquisition, step processing, and UART-based output display. Inter-task communication is achieved using queues, ensuring efficient data flow, while I2C protocol is used for sensor interfacing. This project demonstrates the practical implementation of RTOS concepts such as task scheduling and real-time processing, and highlights how embedded systems can be used to build reliable fitness tracking solutions.
